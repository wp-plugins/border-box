== Description ==

An easy way to place a border box (johnson box) on your Wordpress site.

###Quick Start Guide

* [border box quick start guide at thinklandingpages.com] (http://www.thinklandingpages.com/border-box/)

###What you get when you use the Border Box

*  Place a border box with a simple shortcode [border_box id="%postId%"]
*  Put a border box on any post or page.
*  border box headline
*  border box background color
*  border box message


###Add border box to any post or page

Whether you are writing a post about your company or another company, you can include a well formatted border box.


###Border Box 

Add as many border boxes to your site as you like.  The Border Box lets you create border boxes and save them for when ever you are ready to use them.

###Quick Start Guide

* [border box quick start guide at thinklandingpages.com] (http://www.thinklandingpages.com/border-box/)


== Installation ==


1. Upload `border box` to the `/wp-content/plugin/` directory
1. Activate the border box through the 'Plugins' menu in WordPress
1. Click **Border Box** on the admin menu to enable and set your options.


== Frequently Asked Questions ==

= Do I have to know how to program or design? =

No.  The border box does the programming and design.

= Is there a limit to the number of border boxes I can create? =
No, unlimited border boxes can be created.

= Can the border box be put on my homepage? =
Yes, you can put a border box on your homepage by using the Border Box shortcode on you homepage.

= Can I put border boxes on any post or page? =
Yes, use the border box shortcode, [border_box id="%postId%"] on your pages and posts.
